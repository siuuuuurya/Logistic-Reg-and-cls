# Logistic Regression Models

## Overview
This repository contains machine learning projects built using Logistic Regression for classification and prediction tasks. These projects demonstrate data preprocessing, model training, prediction, evaluation, and GUI integration using Python and Scikit-learn.

## Projects Included

### # Admission Prediction
Predicts the likelihood of student admission based on academic and profile-related features.

### # Admission Prediction (Alternative Version)
An alternative implementation of the admission prediction model for experimentation and comparison.

### # Digit Prediction
Classifies handwritten digits using Logistic Regression.

### # Loan Prediction
Predicts loan approval status using applicant and financial information.

### # Loan Prediction with GUI
A loan prediction system integrated with a graphical user interface (GUI) for user-friendly interaction.

### # SUS Login Prediction
A classification model designed to predict login-related outcomes.
